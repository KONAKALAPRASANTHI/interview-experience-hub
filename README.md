# Interview Experience Hub

A modern MERN-stack platform where students and job seekers can explore interview experiences, company reviews, and job opportunities shared by the community.

## Features

### Home Page
- Browse interview experiences from different companies
- Clean and modern UI
- Responsive cards layout

### Company Reviews
- Read company reviews shared by users
- Like and comment functionality
- Share experiences with the community

### Jobs Portal
- Browse job opportunities
- Search jobs by company or role
- Filter by:
  - Location
  - Job Type
  - Work Mode (Remote / Hybrid / On-site)

### Authentication
- Secure authentication using Clerk
- Sign In / Sign Up
- User Profile Management

### Profile
- Simple user profile section
- Clerk User Management integration

---

## Tech Stack

### Frontend
- React.js
- Vite
- React Router
- Clerk Authentication

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

### Deployment
- Frontend: Render
- Backend: Render
- Database: MongoDB Atlas
